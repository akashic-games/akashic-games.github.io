  • 2021 年 10-12 月
  • 2022 年 1-6 月

更新情報 (〜2022/mm/dd)

# sandbox.config.js で各種警告、表示設定、ウィンドウサイズを設定できるように

akashic-sandbox や akashic serve コマンドの設定ファイル sandbox.config.js で、次の内容を指定できるようになりました。

  • akashic serve の表示オプション (ウィンドウサイズ合わせてゲーム画面を拡大縮小するなど)
  • akashic serve で子ウィンドウを開いた時のサイズ
  • akashic-sandbox, akashic serve の各種警告機能の有効・無効

表示オプション

sandbox.config.js に displayOptions が追加されました。displayOptions の各プロパティの値を設定することにより、serve の実行画面の各種表示を設定できます。

詳細はsandbox.config.js の仕様の displayOptionsを参照してください。

子ウィンドウを開いた時のサイズ

sandbox.config.js に windowSize windowSize を設定することにより、Akashic serve で新規インスタンス表示時のウィンドウサイズを設定できます。

詳細はsandbox.config.js の仕様の windowSizeを参照してください。

各種警告機能の有効・無効

sandbox.config.js に warn が追加されました。warn の各プロパティの値に真偽値を設定することにより、Akashic Sandbox や serve での警告の表示を制御できるようになります。意図的な処理で警告が出てしまう場合に、警告を抑止するために利用できます。 各プロパティのデフォルト値は全てオンとなります。

詳細はsandbox.config.js の仕様の warnを参照してください。

以下の sandbox.config.js の例は、ゲーム画面をウィンドウサイズに合わせ拡大縮小し、子ウィンドウを開いた時のサイズを 640x360 で指定しています。

sandbox.config.js

module.exports = {
  displayOptions: {
    // akashic serve のゲーム画面をブラウザウィンドウに合わせて拡大縮小する
    fitsToScreen: true
  },
  // akashic serve で子ウィンドウを開いた時のサイズを 640x360 に
  windowSize: {
    width: 640,
    height: 360
  }
};

# akashic scan コマンドで出力される game.json のフォーマットを .editorconfig に準拠するように

akashic-cli@2.16.0 以降において、akashic scan コマンドで出力される game.json のフォーマットを .editorconfig に準拠するように変更しました。

# その他の改善・拡張

  • akashic-sandbox や serve で ES2015 以降の構文や機能を利用した場合の警告を廃止。
  • akashic serve で子ウィンドウのデフォルトサイズを、ゲーム画面のサイズに応じて決定するように。
  • sandbox.config.js に server.external プロパティを追加。
