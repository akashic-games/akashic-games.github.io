Akashic Engine

pdi-types

Đây là core library của JavaScript game engine Akashic Engine, cung cấp các nhóm class độc lập với môi trường dành cho người phát triển game.

Cách sử dụng

Repository (Kho lưu trữ) này là thư viện chỉ chứa code độc lập với môi trường. Do vậy, không thể sử dụng đơn lẻ. Để phát triển game, hãy sử dụng Akashic Sandbox và Akashic CLI. Về cách sử dụng chi tiết của Akashic Engine, xin xem trang chính thức

Sử dụng type definitions trong TypeScript

Trường hợp phát triển game bằng TypeScript, có thể sử dụng index.runtime.d.ts của repository này như một type definitions file. Sau khi cài đặt bằng lệnh npm install -D @akashic/akashic-engine, chạy lệnh tsc sử dụng tsconfig.json tham chiếu đến node_modules/@akashic/akashic-engine/index.runtime.d.ts

Cách build

Akashic Engine là JS Module được viết bằng TypeScript. Để build thì cần Node.js. Thực hiện lệnh sau đây trực tiếp từ repository: 

npm install
npm run build

Cách test

 
npm test
