Akashic Engine 入門 (v2版)

はじめに

Akashic Engine は JavaScript で動作するゲームエンジンです。 マルチプラットフォームで動作するゲームを開発できます。 Akashic Engine はオープンソースで開発されているフリーソフトウェアです。 作成したゲームの公開にロイヤリティは必要ありません。

Akashic Engine にはゲーム開発で利用できるツールとライブラリが用意されており、 今すぐゲーム開発を始めることができます。

  • Akashic Engine の現在の実装は2D描画を主体とするゲームに特化しています。
