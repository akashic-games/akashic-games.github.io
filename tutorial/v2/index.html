Akashic Engine 入門 (v2 版)

はじめに

Akashic Engine は JavaScript で動作するゲームエンジンです。マルチプラットフォームで動作するゲームを開発できます。 Akashic Engine はオープンソースで開発されているフリーソフトウェアです。作成したゲームの公開にロイヤリティは必要ありません。

Akashic Engine にはゲーム開発で利用できるツールとライブラリが用意されており、今すぐゲーム開発を始めることができます。

Akashic Engine の現在の実装は 2D 描画を主体とするゲームに特化しています。

このページは Akashic Engine v2 系を対象としています。非推奨の過去版の文書は こちら です。

Akashic Engine の最新バージョンは v3.x.x です。Akashic Engine v2 で作成されたコンテンツを Akashic Engine v3 へ移行するにはこちらを参照してください。
