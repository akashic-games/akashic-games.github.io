game.json 生成ツール

『RPGツクールMV』で作成されたゲームをニコ生ゲームに変換する際の補助ツールです。
ニコ生ゲーム (Akashic Engine 製ゲーム) に必要な設定ファイル「game.json」の内容を自動生成します。

ここに img, audio, data フォルダをコピーした game フォルダをドラッグ＆ドロップしてください。
→

使い方

  1. PRGツクールMVの [ファイル] - [デプロイメント] で 配布用ファイルを作成する。
    • 「プラットフォーム」は「ウェブブラウザ」
    • 「オプション」の「未使用ファイルを含まない」にチェック
  2. RPG ツクール MV ニコ生ゲーム化キットをダウンロードして解凍。以降、tkoolmv-namagame-kit フォルダ中の game フォルダをゲームのテンプレートとして利用していく
  3. 生成された配布用ファイルの img, audio フォルダを、game フォルダの assets フォルダ内にコピーする
  4. 生成された配布用ファイルの data フォルダの中身を、game フォルダの text フォルダ内にコピーする
  5. 上の左の矩形内に、game フォルダをドラッグ＆ドロップする
  6. 上の右の矩形内に game.json の内容が生成される
  7. 「ダウンロード」ボタンを押して game.json をダウンロードし、game フォルダの game.json に上書きコピーする
その他必要な手順や制約は game フォルダの README.md を参照してください。
